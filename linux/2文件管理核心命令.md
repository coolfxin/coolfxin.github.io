# 少年的Linux学习笔记



## 文件管理核心命令



cd

~~~
.                  当前目录
..                 上级目录
/                  跟目录
-                  上一次打开的目录
~~~



ls

~~~
ls                 列出所有文件

ls -l 		       列出文件详细信息
ls -lh             人类视角列出详细信息（此处可写为ls -l -h，h代表human）
ls --full-time     列出详细时间
ls -t              根据最后修改的时间排序
ls -F              在不同文件结尾输出不同特殊符号:
                   /表示文件夹
                   @表示软连接,即快捷方式
                   *表示可执行文件
                   普通文件类型结尾什么都没有
ls -S              针对文件大小进行排序，默认从大到小
ls -r              reverse逆序
ls -i              显示文件的inode信息，相当于文件的身份证号，存储了文件的大小，位置，权限等
~~~



su

~~~
su - 用户名         切换到该用户名的账户
~~~



mkdir

~~~
mkdir              创建文件夹
mkdir {a,b,c,d,e}  生成以大括号中的字符命名的文件夹，每个字符对应一个文件夹
mkdir -p           按照给出的树结构创建一系列文件夹
mkdir fx{1..100}   生成100个文件夹，名字从fx1，fx2一直到fx100

~~~
